# Select Theme - VS Code Extension

## Welcome to Moses Arulvin Theme Manager

Instantly browse, preview, and switch between all your installed VS Code themes with convenient toolbar buttons and a real-time status bar indicator.

---

## ✨ Features

### 🎨 Quick Theme Selection
- **One-click theme picker:** Open VS Code's theme browser instantly from the toolbar
- **Live preview:** See themes in real-time before applying
- **All themes accessible:** Browse all installed themes in one convenient location

### 🌓 Light/Dark Mode Toggle
- **Instant switching:** Toggle between Default Light+ and Dark+ themes with one click
- **Smart auto-detect handling:** Automatically disables OS-based theme switching when you manually toggle

### 🔄 Auto-Detect Control
- **Toggle auto-detect:** Enable/disable OS-based automatic theme switching
- **Visual feedback:** See confirmation when auto-detect is enabled or disabled

### 📊 Status Bar Indicator
- **Live theme display:** Always see your current theme name in the status bar
- **Clickable shortcut:** Click the status bar to open the theme picker
- **Auto-updates:** Status bar updates instantly when themes change

---

## 🎯 How to Use

### Toolbar Buttons (Top-Right Corner)
Three convenient buttons appear in your editor toolbar:

1. **🔄 Auto-Detect Toggle** - Enable/disable automatic theme switching based on OS appearance
2. **🌓 Light/Dark Toggle** - Switch between Default Light+ and Default Dark+ themes
3. **🎨 Theme Picker** - Open the full theme selection menu

### Status Bar (Bottom-Right)
- **View current theme:** See your active theme name at all times
- **Click to select:** Click the status bar item to open the theme picker

### Command Palette
Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac) and search for:
- `Select Theme` - Open theme picker
- `Toggle Mode (Light/Dark)` - Switch between light and dark
- `Toggle Auto-Detect Color Scheme` - Control OS-based theme switching
