# Select Theme

A simple and convenient VS Code extension that allows you to quickly browse and switch between all your installed color themes.

## Features

- **Quick theme switching**: Click the theme icon in the editor title bar to open a dropdown with all available themes
- **Easy to access**: Theme selector button appears in the top-right corner of the editor
- **Current theme indicator**: The currently active theme is marked with a checkmark
- **Alphabetically sorted**: All themes are sorted alphabetically for easy browsing

## Usage

1. Click the color palette icon (🎨) in the top-right corner of the editor
2. Browse through the list of available themes
3. Click on any theme to apply it immediately

You can also access the theme selector through the Command Palette:
- Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac)
- Type "Select Theme"
- Press Enter
